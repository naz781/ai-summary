# AI Summary Web App

A web application that leverages the OpenAI API to generate AI-powered summaries, similar to QuiltBolt. Built with React.js and Firebase for authentication.

## Features
- User authentication with **Firebase** (sign up/login)
- AI-powered summary generation using **OpenAI API**
- Modern and responsive UI built with **React.js** and **Tailwind CSS**
- Interactive and user-friendly design

## Tech Stack
- **Frontend:** React.js, Tailwind CSS, React Router DOM, React Icons
- **Backend/Auth:** Firebase Authentication
- **AI Integration:** OpenAI API
- **Email Integration:** EmailJS (for contact forms)


## Installation

1. Clone the repository:

```bash
git clone <repo-url>
```
## Navigate into the project folder:
```bash
cd summaryopenai
```
### Create a .env file in the root and add your API keys:
```bash
REACT_APP_OPENAI_KEY=your_openai_api_key
REACT_APP_FIREBASE_API_KEY=your_firebase_api_key
REACT_APP_FIREBASE_AUTH_DOMAIN=your_firebase_auth_domain
REACT_APP_FIREBASE_PROJECT_ID=your_firebase_project_id
REACT_APP_FIREBASE_STORAGE_BUCKET=your_firebase_storage_bucket
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your_firebase_sender_id
REACT_APP_FIREBASE_APP_ID=your_firebase_app_id
```
### Start the development server:
```bash
npm start
```
### Usage

Sign up or log in using Firebase Authentication

Generate AI summaries using the OpenAI API

Use the contact form powered by EmailJS (optional)
